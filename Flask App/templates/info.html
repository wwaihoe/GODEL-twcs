GODEL

GODEL (Grounded Open Dialogue Language Model) Microsoft GODEL is a large-scale pre-trained model for goal-directed dialogs that uses a Transformer-based encoder-decoder model. The model can be fine-tuned to generate task-specific dialog for new tasks through training data on task-specific dialog.
GODEL Project

GODEL_twcs

Pre-trained model was fine-tuned using dialog between customers and customer support on Twitter for tech-related companies
Dataset
List of accounts used for fine-tuning: "hulu_support", "sprintcare", "comcastcares", "XboxSupport", "VerizonSupport", "TMobileHelp", "SpotifyCares", "MicrosoftHelps", "HPSupport", "DellCares", "Ask_Spectrum", "AskPlayStation", "AppleSupport", "AdobeCare"

GODEL_twcs_AppleSupport

Pre-trained model was fine-tuned using dialog between twitter users and Apple's customer support account on Twitter

GODEL_twcs_SpotifyCares

Pre-trained model was fine-tuned using dialog between twitter users and Spotify's customer support account on Twitter
